## 关于网站

网站使用 Docusaurus 构建。这个网站的最初目的：为了放置一个民法总则教程（现名《奶龙都能看懂的民法总则手册》）。

网站的架构：

[lizhenyu.top](https://lizhenyu.top)这个域名，我还在考虑要不要续费。[li-zhenyu.pages.dev](https://li-zhenyu.pages.dev)是免费的，会长期使用。但是国内访问可能不太稳定，SEO 效果也不怎么样。
